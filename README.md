## Running the application locally

```bash
mvn clean package

mvn spring-boot:run
```
Swagger UI : http://localhost:8082/swagger-ui/index.html

## Running the application in docker environment

```bash
docker compose up -d
```
Swagger UI : http://localhost:8081/swagger-ui/index.html
